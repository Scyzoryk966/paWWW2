Zmienne

Żródło: robert.type.pl

W języku JavaScript można wykonywać operacje nie tylko na liczbach, ale także na innych typach wartości. Pełna lista tych typów jest następująca (w nawiasach są nazwy angielskie):

  • liczby (Number);
  • ciągi znaków (String);
  • wartości logiczne (Boolean);
  • obiekty (Object);
  • tablice (Array);

oraz dwa typy/wartości specjalne: Null i Undefined. Wykonamy nastepujący program:

<script language="javascript">
        var imie = "Damian"
        var nazwisko = "Szefler"
        var wiek = "23"
        var napis = "Nazywasz się " + imie + " " + nazwisko + " i masz " + wiek + " lat.";
        document.write(napis);
</script>
